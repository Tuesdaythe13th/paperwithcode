resources

A curated repository of research, tools, and operational frameworks across AI safety, agentic systems, and socio-technical risk.

To help you navigate this massive repository of research, tools, and operational frameworks, I have organized the resources into eight strategic clusters.

I. Frontier AI Research & Multimodal Models

Next-generation architectures focusing on visual reasoning, thinking loops, and mechanistic interpretability.

  • ERNIE-4.5-VL-Thinking (Baidu): 28B parameter model featuring “Thinking with Images” and visual grounding. Model Artifacts (HuggingFace) | Baidu Blog Post
  • CWM: Code World Model (Meta/Facebook): 32B open-weights LLM for code generation and software reasoning. Model artifacts (HuggingFace) | GitHub Repository
  • Neuronpedia: A platform for exploring and visualizing the internal features (neurons) of LLMs. Home Site
  • SynthID Text: Google’s toolkit for watermarking and identifying AI-generated text content. Technical Overview
  • ArxivLens: Research paper analysis tool for deep cognitive architecture reviews.
  • LLM Comparator: A tool for side-by-side model analysis.

II. Agentic AI & Reinforcement Learning (RL)

Resources dedicated to autonomous agents that plan, self-correct, and learn through environment interaction.

  • SEAL: Self-Adapting Language Models that refine themselves through continual intelligence. GitHub - Continual-Intelligence/SEAL
  • AgentRL: Framework for scaling agentic reinforcement learning across multi-turn tasks. Paper & Project Details
  • SALT: Step-level advantage assignment for long-horizon agents via trajectory graphs. Technical Paper
  • Model Context Protocol (MCP) Inspector: A testing tool for the Model Context Protocol used in agentic reasoning. MCP GitHub
  • Agent Learning via Early Experience: Improving training efficiency.
  • In-the-Flow Optimization: Real-time agentic system planning and tool use.

III. Robotics & Physical Embodiment

Integrating AI intelligence into humanoid hardware and physical simulation environments.

  • Atlas & Atlas MTS (Boston Dynamics): Mobile manipulation using Diffusion Transformer-based policies. Full Technical Analysis
  • MuJoCo: Multi-Joint dynamics with Contact; the industry-standard physics engine for robotics. MuJoCo Documentation
  • Gemini Robotics 1.5: Google’s latest framework for deploying visual agents into physical worlds. Google Blog Post

IV. Environmental Mapping & Geospatial Data

Platforms for planetary-scale monitoring, ecosystem classification, and biodiversity analysis.

  • Google Earth Engine: Cloud platform for satellite imagery and geospatial analysis. Python/JS API GitHub
  • Microsoft Planetary Computer: Cloud access to petabytes of environmental data. Planetary Computer Home
  • Collect Earth Online (CEO): Open-source high-resolution imagery monitoring tool. CEO Home | Online Documentation
  • IUCN Global Ecosystem Typology: A standard classification for global biomes. IUCN Typology Home
  • Cartopy: Python library for geospatial data processing and map production. Cartopy Documentation

V. Neuroscience & Biological Research

Foundational research tools for mapping the brain and tracking biomedical research funding.

  • Allen Brain Cell Atlas: Programmatic access to the Human Whole-Brain Transcriptomic Atlas. ABC Atlas Access GitHub
  • NIH RePORT: Database for searching National Institutes of Health research projects and funding. NIH RePORTER

VI. Developer Tooling & AI Evaluation

Toolkits for trace grading, performance measurement, and proactive risk identification.

  • Stax: Google’s complete toolkit for generative AI evaluation.
  • PyRIT (Azure): Open-source Python Risk Identification Tool for proactive AI red-teaming. GitHub - Azure/PyRIT
  • FiftyOne: Tool for visualizing datasets and computer vision models (includes Gemini integration). FiftyOne Documentation
  • OpenAI Agent Cookbook: Step-by-step guides for building coding agents and guardrails. OpenAI Cookbook GitHub
  • Inspect: A scanning and evaluation framework. Inspect Repository

VII. Business, Strategy & Governance

Professional curriculum, contract intelligence, and global AI policy mapping.

  • HBS Online Syllabi: Negotiation Mastery | Launching Tech Ventures | Entrepreneurship Essentials
  • MIT AI Governance Map: Interactive visualization of global AI laws and regulations. MIT AI Governance Tracker
  • Workday Contract Intelligence: AI-native contract lifecycle management. CLM Datasheet | Intelligence Datasheet
  • Common Paper: Open-source standardized commercial contracts. Common Paper Home
  • A Pragmatic View of AI Personhood: Philosophical frameworks for autonomous software. Research Paper
  • Fear and Greed Index: Market sentiment metrics impacting AI capital. CNN Business Index

VIII. Forecasting & Reproducibility

Platforms and studies focusing on the future of research and prediction markets.

  • Polymarket: Decentralized prediction market for global events. Polymarket Home
  • PredictIt: Research project for trading on political and economic outcomes. PredictIt Home
  • PNAS Reproduction Study: Research on using prediction markets to estimate scientific reproducibility. PNAS Study Link
